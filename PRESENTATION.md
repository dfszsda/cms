# Project 1: College Connect (CMS)

A comprehensive **College Management System** designed to digitize academic workflows.

### Key Highlights:
- **Role-Based Access**: Specialized views for Admin, Student, Teacher, and Retailer.
- **Real-time Synchronization**: Powered by Firebase.
- **Modern UI**: Clean and intuitive design using the Poppins font.

---

# CMS: Detailed Modules

- **Admin Module**: Branch setup, Result management, Library stock, and **UFM Dashboard** for exam discipline.
- **Student Module**: Attendance tracking, Exam forms, Results, and Canteen ordering.
- **Retailer Module**: Real-time order management for the campus canteen.
- **Faculty Module**: Management of student lists and academic records.

# Project 2: Gokudiyugam

A spiritual and administrative platform for **BAPS Badalpur Mandal**.

### Core Philosophy:
Bridging the gap between **Tradition and Technology** to keep the community connected.

---

# Gokudiyugam: Key Features

- **Daily Darshan**: Fast-loading, high-quality images via the 'Coil' library.
- **Kirtan Player**: Full-featured media player with categorized content.
- **Smart Search**: English-to-Gujarati transliteration for lyrics.
- **Sabha Management**: Digital summaries (Sabha Saar) and weekly timetables.
- **Personal Touch**: Automated Birthday Wishes for all users.

---

# Technical Architecture

- **Frontend**: Flutter (Cross-platform Android & iOS).
- **Backend**: Firebase (Auth, Firestore, Storage).
- **Intelligence**: Google Gemini 1.5 Flash for AI-driven content processing.
- **Security**: Biometric Authentication & Firebase App Check.
