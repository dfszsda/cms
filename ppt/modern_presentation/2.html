Git Core Commands

git init

Initialize a new local Git repository.

git clone [url]

Create a copy of a remote repository.

git status

Show status of changes (untracked, modified).

git add .

Stage all changes for the next commit.

git commit -m "msg"

Create a permanent snapshot of changes.

git push origin main

Upload local changes to remote server.
